# 💰 FinTracker – Smart Personal Finance & Expense Management System

FinTracker is a cloud-based personal finance management system developed using Amazon Web Services (AWS). It helps users manage their monthly income, expenses, savings, and budget through a simple and user-friendly web application.

---

## 📌 Project Overview

Managing personal finances manually can be difficult and time-consuming. FinTracker provides an easy way to:

- Record monthly salary
- Set savings goals
- Track daily expenses
- Categorize expenses
- View monthly financial summaries

The application uses a serverless AWS architecture, making it scalable, secure, and cost-effective.

## ⚙️ How to Run

1. Download or clone this repository.

2. Open the project folder.

3. Open `index.html` in your web browser.

4. Configure the AWS API Gateway endpoint inside `js/script.js`.

5. Start tracking your income and expenses.
